              ZAMAN BARBERSHOP — ВОЗМОЖНОСТИ СИСТЕМЫ
═══════════════════════════════════════════════════════════


───────────────────────────────────────────────────────────
  АДМИНИСТРАТОР
───────────────────────────────────────────────────────────

▌ ЗАПИСИ (раздел "Записи")
  • Просматривать записи по любой дате через календарь
  • Видеть процент заполненности каждого дня (SVG-кольцо)
  • Подтверждать запись клиента — клиенту приходит уведомление
  • Отменять запись с указанием причины — клиенту приходит уведомление
  • Написать клиенту напрямую в Telegram (кнопка ✉️)
  • Видеть статус каждой записи: Ожидает / Подтверждён / Отменён
  • Видеть пожелания клиента к записи (комментарий)

▌ ОПЛАТЫ (раздел "Оплаты")
  • Видеть список записей, ожидающих подтверждения предоплаты
  • Подтверждать получение предоплаты — клиенту приходит уведомление
  • Отменять запись прямо из раздела оплат

▌ КЛИЕНТЫ (раздел "Клиенты")
  • Просматривать всех клиентов с аватаркой из Telegram
  • Видеть количество визитов и общую сумму трат каждого клиента
  • Видеть дату последнего визита
  • Блокировать клиента (с причиной) — заблокированный не может записаться
  • Разблокировать клиента
  • Написать клиенту напрямую в Telegram

▌ УСЛУГИ (раздел "Услуги")
  • Добавлять новые услуги (название, цена, длительность)
  • Удалять услуги

▌ ГРАФИК РАБОТЫ (раздел "График")
  • Включать/выключать рабочие дни недели (Пн–Вс)
  • Устанавливать время начала и конца рабочего дня для каждого дня
  • Выбирать интервал между слотами: 30 / 45 / 60 / 90 минут

▌ НАСТРОЙКИ (раздел "Настройки")

  ┌ ПРОФИЛЬ МАСТЕРА ──────────────────────────────────────┐
  │ • Имя мастера — отображается в шапке клиентского      │
  │   приложения и в заголовке панели управления          │
  │ • Описание (Bio) — подзаголовок в профиле             │
  │ • Адрес / Локация — отображается под профилем         │
  └───────────────────────────────────────────────────────┘

  ┌ КНОПКА ОПЛАТЫ ────────────────────────────────────────┐
  │ • Текст кнопки — например "Оплатить через Kaspi"      │
  │ • Ссылка (URL) — ссылка на страницу оплаты Kaspi      │
  └───────────────────────────────────────────────────────┘

  ┌ УВЕДОМЛЕНИЯ ──────────────────────────────────────────┐
  │ • Уведомления в Telegram — вкл/выкл получение         │
  │   сообщений о новых записях и оплатах                 │
  │ • Запись на сегодня — вкл/выкл экстра-уведомление     │
  │   когда клиент записывается на текущий день            │
  │ • Напоминания клиентам — вкл/выкл автоматические      │
  │   напоминания за день (в 10:00) и за час до записи    │
  └───────────────────────────────────────────────────────┘

  ┌ ПРЕДОПЛАТА ────────────────────────────────────────────┐
  │ • Требовать предоплату — вкл/выкл обязательный аванс  │
  │ • Процент предоплаты — какой % от суммы платит клиент │
  │   (например 30% → запись на 5000₸ = 1500₸ аванс)     │
  └───────────────────────────────────────────────────────┘

▌ ВЫРУЧКА (главная панель)
  • Видеть выручку за текущий месяц
  • Нажать на карточку → список всех оплаченных записей:
    имя клиента, дата, время, услуга, сумма, телефон

▌ КОМАНДЫ БОТА (через Telegram)
  /start         — открыть панель управления
  /set_photo     — установить фото для сообщения о записи
                   (отправить фото с подписью /set_photo)
  /set_ban_photo — установить фото для уведомления о блокировке
  /stats         — краткая статистика в чат
  /block_date    — заблокировать конкретную дату (нет записей)

▌ УВЕДОМЛЕНИЯ АДМИНИСТРАТОРУ ПРИХОДЯТ:
  • Новая запись от клиента
  • Клиент сообщил об оплате предоплаты
  • Клиент отменил запись
  • Запись на СЕГОДНЯ (если включено)

▌ МНОГОАДМИННОСТЬ
  • Поддерживается несколько администраторов
  • Уведомления приходят всем администраторам
  • Каждый администратор имеет свой токен входа


───────────────────────────────────────────────────────────
  КЛИЕНТ
───────────────────────────────────────────────────────────

▌ ЗАПИСЬ
  • Выбрать одну или несколько услуг (с ценой и длительностью)
  • Выбрать дату из доступных (доступные дни отмечены зелёной точкой)
  • Выбрать удобное время из свободных слотов
  • Написать пожелания к прическе (необязательно, можно пропустить)
  • Подтвердить запись с указанием имени и телефона
  • Имя автоматически подставляется из профиля Telegram

▌ ПРЕДОПЛАТА (если включена администратором)
  • Увидеть сумму предоплаты после выбора услуг
  • Перейти на страницу оплаты через Kaspi (или другой банк)
  • Нажать "Я оплатил(а)" — администратор получает уведомление

▌ МОИ ЗАПИСИ
  • Просматривать свои активные записи
  • Видеть дату, время, услугу, сумму и статус каждой записи
  • Отменить запись самостоятельно

▌ УВЕДОМЛЕНИЯ В TELEGRAM
  • После записи — сообщение "Запись принята и в обработке"
  • После подтверждения администратором — сообщение меняется
    на "Запись подтверждена" с датой, временем и суммой предоплаты
  • При отмене администратором — сообщение с причиной
  • Напоминание за день до записи (если включено)
  • Напоминание за час до записи (если включено)

▌ ОГРАНИЧЕНИЯ
  • Заблокированный клиент не может создавать записи
  • Нельзя записаться на уже занятое время
  • Нельзя записаться на прошедшую дату или выходной день
